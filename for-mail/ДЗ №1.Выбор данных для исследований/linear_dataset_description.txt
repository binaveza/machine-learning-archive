Данные для решения задач регрессии - предсказания непрерывного значения целевой переменной

Датасет House Prices: Advanced Regression Techniques доступен на платформе Kaggle по следующей ссылке:

https://www.kaggle.com/competitions/house-prices-advanced-regression-techniques

Описание данных
Этот датасет содержит информацию о домах, проданных в городе Эймс (штат Айова) с 2006 по 2010 год. Целью является предсказание цены дома (SalePrice) на основе различных характеристик недвижимости.

Описания файлов
train.csv — обучающий набор
test.csv — тестовый набор
data_description.txt — полное описание каждого столбца
sample_submission.csv — эталонная отправка из линейной регрессии по году и месяцу продажи, площади участка и количеству спален

Поля данных

SalePrice — цена продажи недвижимости в долларах. Это целевая переменная для предсказания
